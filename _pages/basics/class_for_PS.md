# Class 

클래스는 코딩 테스트 문제를 풀 때 많이 사용되곤 하는데, 매직 메서드 (magic methods)란, 생성되는 클래스의 인스턴스가 기본적으로 갖춰야 하는 기능을 지정하는 특별한 메서드이며 메서드의 이름이 밑줄 두 개로 감싸진다. 

정의된 클래스에 기본으로 포함된 매직 메서드 목록은 `dir() 함수를 이용하여 확인할 수 있다. 매직 메서드 중에서 코딩 테스트를 풀 때 유용하게 사용되는 매직 메서드에 대해서 알아보자.
